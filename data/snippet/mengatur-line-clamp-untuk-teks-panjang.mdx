---

## âš™ï¸ Contohnya

Pastikan plugin `@tailwindcss/line-clamp` sudah aktif di konfigurasi kamu:

```js
// tailwind.config.js
module.exports = {
  theme: {
    extend: {},
  },
  plugins: [require('@tailwindcss/line-clamp')],
}
```

Lalu tinggal pakai di elemen teks:

```html
<p class="line-clamp-2 text-zinc-700">
  Ini contoh teks panjang banget yang bakal otomatis dipotong setelah dua baris.
</p>
```
